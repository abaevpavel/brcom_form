# Исправление ошибки деплоя на Netlify

## Проблема

При деплое на Netlify возникала ошибка:
```
Error: error:0308010C:digital envelope routines::unsupported
```

## Причина

- **Раньше**: Netlify использовал старую версию Node.js (16 или ниже), которая была совместима с `react-scripts 4.0.3`
- **Сейчас**: Netlify обновился и по умолчанию использует Node.js 22+
- **Проблема**: Старый `react-scripts 4.0.3` использует старый webpack, который не совместим с Node.js 17+

## Решение

Добавлен файл `netlify.toml` с явным указанием версии Node.js 16, которая совместима с проектом:

```toml
[build.environment]
  NODE_VERSION = "16"
```

Теперь Netlify будет всегда использовать Node.js 16 для этого проекта, независимо от обновлений платформы.

## Альтернативное решение

Вместо фиксации версии Node.js можно было бы обновить `react-scripts` до версии 5+, но это требует проверки совместимости всех зависимостей.
